# Unused AstroWind files

This folder is for keeping old template/demo files that may be useful later but should not be part of the live Astro site.

Keep archived files outside `src/pages` so Astro does not create public routes for them.

The files here are the non-duplicate remainder of the AstroWind template archive after shared deployment, editor, and agent-reference files were moved into the matching `extra/` folders.

Files removed during cleanup included old AstroWind demo home pages, landing pages, blog demo pages, blog components, and demo blog posts. They were not kept in the active source tree because the live MathsTutorHelp site does not use them.
